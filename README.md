# Maven
### [Maven中央库](https://search.maven.org/artifact/cn.wxxsxx/sms-sdk-java/4.0.0/jar)

在项目的pom.xml的dependencies中加入以下内容:
```
<dependency>
    <groupId>cn.wxxsxx</groupId>
     <artifactId>sms-sdk-java</artifactId>
     <version>4.0.0</version>
</dependency>
```
